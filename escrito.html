Arquitectura cliente servidor y Modelo Vista Controlador


Arquitectura cliente servidor


The client and server is a software architecture model which consists of two parties, one the server and another is the clients which is connected by a computer network, the client makes requests to a program and the server is the one who responds, for example when we are going to enter YouTube or Facebook, you are the client and the server is Facebook or YouTube

In this model, the client is usually workstations that request services from the server, while a server acts as a data warehouse and functions as a data manager. This model consists of several components, such as network, client, server, protocols, services, database

Modelo Vista Controlador


It is a software architecture pattern that uses 3 components view models and controllers, separates the data and logic of an application from its representation and the module responsible for managing events and communications

The model is responsible for managing all access to information is the representation of the information with which it is operating system

The controller is the intermediary between the view and the model since it is the one that responds to events normally are user actions and makes requests to the model when information requests are made

the view represents the model, this means that it represents the user interface, it is the information that must represent as output
