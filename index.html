Etiquetas HTML
html: indica el comienzo del documento HTML head: indica que empieza la cabecera de la página.
body: es el cuerpo de la página, donde va lo que se ve en el navegador al cargar una web b: se utilizan para resaltar el texto
h1: son los títulos o encabezados a: define los enlaces.
table: es una tabla, y dentro de esta tenemos filas p: texto dentro de esta etiqueta forma un párrafo.
img: imágenes ul: los textos dentro de esta etiqueta se estructuran en listas

Estilos CSS
Colors: La propiedad color establece el color del texto / sintaxis: { color: lime; } Backgrounds: Establece algunas o todas las propiedades del color y/o imagen de fondo de un elemento / sintaxis: background: #CCC; ,background-color: #CCC; background-image .
borders: Establece algunas o todas las propiedades de todos los bordes de los elementos / sintaxis: { border: 1px solid #C00; } margins: establece el margen para los cuatro lados / sintaxis: margin-top: 10px; margin-right: 10px;
text:se puede usar para colocar texto de diferentes maneras / sintaxis: text-decoration, text-transform: uppercase; fonts: permite establecer de una sola vez los valores para todas las propiedades / sintaxis: font-style, font-variant, font-weight, font-size, line-height y font-family.
Height: eespecifica la altura del area de contenido de un elemento / sintaxis: height: auto; height: 120px; height: 10em; width: especifica la anchura del area de contenido de un elemento / sintaxis: . width: 300px;
links: representa un elemento que aún no se ha visitado / sintaxis a:link { background-color: gold; color: green; } tables: representa un elemento en el cual se puede modificar las tablas de la etiqueta table en el codigo ya escrito / sintaxis: table { width: 100%; } th, td { width: 25%; }
position: especifica cómo un elemento es posicionado en el documento / sintaxis: Overflow: los textos dentro de esta etiqueta se estructuran en listas position: realtive; position: absolute; position: fixed;
float: especifica si un elemento debe salir del flujo normal y aparecer a la izquierda o a la derecha de su contenedor, donde los elementos de texto y los en línea aparecerán a su alrededor. / sintaxis: float: left | right | none | inherit
